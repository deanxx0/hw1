10. 입력 폼 만들기

형식 구성 내용
<FORM> <FORM>...</FORM> 양식 정의.
<FORM METHOD="..." ACTION="..."> 양식의 방법 정의.
<TEXTAREA> <UTEXTAREA NAME="..." ROWS="..."> 글 틀 상자 정의.
<INPUT> <INPUT TYPE="..."> 어떤 형태로 입력할 것인지 정의.
<INPUT TYPE="RADIO"> 라디오 단추 만들어준다.
<INPUT TYPE="CHECKBOX"> 체크 상자를 만들어준다.
<INPUT TYPE="SUBMIT"> 제출 버튼을 만들어준다.
<INPUT TYPE="RESET"> 취소 버튼을 만들어준다.
<SELECT> <SELECT NAME="..."> 선택사항 상자를 만들어준다.
